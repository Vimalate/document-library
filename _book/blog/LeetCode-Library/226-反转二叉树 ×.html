226-反转二叉树 ×

    • 原题
    • 解 :

原题

翻转一棵二叉树。

示例：

输入：

     4
   /   \
  2     7
 / \   / \
1   3 6   9
输出：

     4
   /   \
  7     2
 / \   / \
9   6 3   1

解 :

var invertTree = function (root) {
    if (!root) {
        return null
    } else {
        return {
            //递归遍历，先从根结点开始，左右交换
            val: root.val,
            left: invertTree(root.right) || null,
            right: invertTree(root.left) || null
        }
    }
};
